# Projek CLO2 - Keamanan Sistem Informasi

Proyek web sederhana dengan implementasi keamanan.

## Branches
- `tanpa-keamanan` - Versi tanpa implementasi keamanan
- `dengan-keamanan` - Versi dengan implementasi keamanan lengkap

## Cara Menjalankan
```bash
git checkout <nama-branch>
docker compose up -d --build
```
